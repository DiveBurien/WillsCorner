## 📗 CCNAv7: Switching, Routing, and Wireless Essentials (SRWE)
**Class:** cis161 fall 23 | **Instructor:** Chun Yu | **Completed:** Dec 2023

The second course in the CCNAv7 curriculum, covering 16 modules of advanced switching, routing, and wireless networking concepts with hands-on labs using Cisco Packet Tracer. Topics covered include:

### 🔀 Switching Concepts & VLANs (Modules 1–4)
- Layer 2 switching operation — MAC address tables, frame forwarding, CAM table management
- Collision domains vs. broadcast domains
- Switch initial configuration and secure management access
- VLAN concepts, implementation, and configuration on Cisco switches
- VLAN trunking — 802.1Q encapsulation and trunk port configuration
- Dynamic Trunking Protocol (DTP) configuration and security implications
- Inter-VLAN routing — router-on-a-stick and Layer 3 switching methods
- Troubleshooting inter-VLAN routing on Layer 3 devices

### 🔁 Redundant Networks (Modules 5–6)
- Spanning Tree Protocol (STP) concepts and operation
- Rapid PVST+ (Rapid Per-VLAN Spanning Tree Plus) configuration and troubleshooting
- EtherChannel technology — LACP and PAgP configuration
- Troubleshooting EtherChannel on switched links

### 📡 Available & Reliable Networks (Modules 7–9)
- DHCPv4 — server configuration, relay agents, and troubleshooting
- DHCPv6 — stateless (SLAAC) and stateful DHCPv6 configuration
- HSRP (Hot Standby Router Protocol) — first-hop redundancy for gateway availability
- Virtual IP addresses, virtual MAC addresses, and failover behavior

### 🔒 Layer 2 Security & WLANs (Modules 10–13)
- Layer 2 security threats — MAC flooding, VLAN hopping, DHCP spoofing, ARP poisoning
- Switch security best practices — port security, DHCP snooping, Dynamic ARP Inspection (DAI)
- Wireless LAN (WLAN) concepts — 802.11 standards, SSID, BSS, ESS
- Wireless security protocols — WPA2, WPA3, and enterprise authentication
- WLAN configuration on Cisco wireless routers and access points
- Wireless troubleshooting

### 🌐 Routing Concepts & Configuration (Modules 14–16)
- Routing concepts — routing table structure, administrative distance, longest prefix matching
- Static and default route configuration (IPv4 and IPv6)
- Floating static routes for backup path redundancy
- Troubleshooting static and default routes
- IPv6 routing configuration and verification
