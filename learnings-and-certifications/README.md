### CCNAv7: Introduction to Networks
Covered the foundational concepts of networking including the OSI and TCP/IP models, IP addressing and subnetting, Ethernet and switching basics, routing fundamentals, and basic network security concepts. This course established the networking knowledge base that supports all subsequent security work.

### CCNAv7: Switching, Routing, and Wireless Essentials
Built on the Introduction to Networks course with deeper coverage of VLANs, inter-VLAN routing, STP (Spanning Tree Protocol), EtherChannel, DHCPv4/v6, SLAAC, HSRP, and wireless networking concepts including WLANs and security protocols. This course developed the hands-on routing and switching skills directly applicable to understanding network topology, segmentation, and traffic flow in security contexts.
