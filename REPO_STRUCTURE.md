# Repository Structure

```
pentest-report/
├── README.md              # Full penetration test report (rendered on GitHub)
├── REPO_STRUCTURE.md      # This file
├── docs/
│   └── Schnaith-1_FinalReport_CIS450.docx   # Original Word document
└── artifacts/
    └── (place screenshots here, named Figure-1.png, Figure-2.png, etc.)
```

## How to Use This Repo

1. The `README.md` is the primary report — it renders automatically on GitHub.
2. Drop your screenshots into the `artifacts/` folder, named to match the figure references in the README (e.g., `Figure-3.png`).
3. Update the README artifact tables to link to those images using:
   ```markdown
   ![Figure 3](artifacts/Figure-3.png)
   ```
